Elemento de bloco e de linha

Elementos de bloco

Sempre ocupam toda a largura do elemento pai. Quando colocado em sequência, ficam um embaixo do outro.

p

div

h1

blockquote
  1. ol

Elementos de linha

Ocupam apenas a largura necessária para o seu conteudo e ficam lado a lado caso haja largura disponivel

a span em strong code abbr

Mudando o comportamento

Um elemento bloco pode se comportar como um elemento de linha colocando, no CSS display : inline.

div em linha

p em linha

Da mesma forma , um elemento dae linha pode se comportar como um elemento de bloco colocando-se, no CSS, display: block.

span em blocos code em blocos
